A C# Windows Forms application developed during the DecodeLabs Cybersecurity Internship.

## Features
- Password length validation
- Uppercase letter detection
- Number detection
- Symbol detection
- Password strength classification:
  - Weak
  - Medium
  - Strong

## Technologies Used
- C#
- Windows Forms
- Visual Studio 2022

## Purpose
This application helps users evaluate password strength and promotes secure password practices.
